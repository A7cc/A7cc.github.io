DELL服务器安装ESXI

2023/3/14 服务器 ESXI

壹 DELL服务器（工作站）安装ESXi

本安装环境所需ESXi镜像：6.5 版本

贰 安装

首先使用UltaISO进行U盘刻录（前提），接着插入U盘，设置为U盘启动，进入引导（前面这些基本上是服务器的引导界面做的，不仔细说明，包括Reid，引导）

假设我们已经把前面的步骤搞完，也就是可以正常进入ESXI安装界面，如下：


出现一系列页面后，出现以下界面，按Enter确认安装：

F11 接受协议并继续：

此界面为选择安装位置，Enter 确认，安装到本地磁盘中：

键盘类型，保持默认美式键盘，Enter 继续：

输入root密码，密码要求7位以上，按 Enter 继续（Tab是换行）：

选择F11 安装：

选择第一行，重新开机，这时候记得把U盘取出来，否则会重新安装之前的步骤：

至此安装完成，进入控制台，通过控制台可以看到服务器的基本信息以及IP地址，可以看到IP为0.0.0.0，不可用需要配置网络，看叁配置！

叁 配置

按F2进入配置

配置页面英文 配置页面翻译
ConfigurePassword 配置root密码
ConfigureLockdown Mode 配置锁定模式。启用锁定模式后，除vpxuser以外的任何用户都没有身份验证权限，也无法直接对ESXi执行操作。锁定模式将强制所有操作都通过vCenter Server执行
Configure Management Network 配置网络
Restart Management Network 重启网络
Test Management Network 使用Ping命令测试网络
Network Restore Options 还原网络配置
Configure Keyboard 配置键盘布局
Troubleshooting Options 故障排除设置
ViewSystem Logs 查看系统日志
ViewSupport Information 查看支持信息

3.1 更改密码

选中第一项Configure Password，按Enter确认，提示输入老密码和二次新密码，按Enter，密码修改完成：

3.2 网络设置

设置静态IP地址由于IP地址默认是DHCP获取的，企业环境中都要配置静态IP地址，选中Configure Management Network按“回车”键确认，选择IPV4

网络配置页面英文 网络配置页面翻译
NetworkAdapters 网卡选择
VLAN（optional） 设置VLan
IPv4Configuration 设置IPv4地址
IPv6Configuration 设置IPv4地址
DNSConfiguration 设置DNS地址
CustomDNS Suffixes 自定义DNS后缀

按空格键选中第三项，设置静态IP地址、掩码、网关的信息

配置页面英文 配置页面翻译
DisableIPv4 configuration for management network 禁用IPv4地址
Usedynamic IPv4 address and network configuration 配置动态IPv4地址
Setstatic IPv4 address and network configuration 配置静态IPv4地址

选中DNS Configuration，按Enter键确认，可根据实际需要更改DNS和本机名，按Enter键确认：

肆 密钥

配置完网络后，重启服务器，重启后使用浏览器访问ESXI，登录，选择管理 —=> 许可 -=> 分配许可证，许可证：
